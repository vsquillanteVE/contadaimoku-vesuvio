Contadaimoku Vesuvio API

Benvenuti all'API del progetto Contadaimoku Vesuvio.

Endpoints disponibili:

GET /api/count

Restituisce il conteggio attuale del Daimoku.

POST /api/count

Incrementa il conteggio del Daimoku.

Body: { "amount": number }

GET /api/message

Restituisce il messaggio in evidenza.

POST /api/message

Aggiorna il messaggio in evidenza (richiede autenticazione).

GET /api/message/history

Restituisce la cronologia dei messaggi.

POST /api/login

Effettua il login come amministratore.

Per utilizzare questa API, fare riferimento alla documentazione completa.
